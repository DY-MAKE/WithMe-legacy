# 위드미(WithMe)

위드미는 인공지능 기반 시각 장애 보조 어플리케이션 서비스입니다.


> [!Warning]
> 이 프로젝트는 [withme 레포지토리](https://github.com/DY-MAKE/withme/)에서 다시 개발중입니다.
